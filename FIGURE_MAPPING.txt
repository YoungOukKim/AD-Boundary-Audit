Added analyses (R/analysis/):
  boundary_detection.R                -> standalone 9-algorithm/19-component engine (set.seed 42)
  synthetic_benchmark.R               -> Figure 7 ground-truth benchmark: plants known boundaries in synthetic
                                         marker x bin matrices; applies the SAME engine + audit gates (consensus
                                         support, random-panel specificity, leave-one-marker-out) under 5 regimes
                                         (real / global-offset / null / ramp / single-marker decoy). A-priori gates
                                         (S*=95th pct null support; p<=0.05; LOMO<=1 bin). NPERM=300, B=20, K=44.
                                         -> output/figures/Figure7_benchmark.png + output/tables/benchmark/{partA.csv, global.csv, lomo.csv, summary.txt}
  verify_correction_consistency.R     -> HK-flat & METAB-conservation under one genome-wide correction
                                         input: data/metabolic_correction/panel_correction_z.csv
  external_marker_audit.R             -> adjudicates external candidate markers (NPTX1/NPTXR/NPTX2/NRGN/SNAP25/VAMP2)
                                         from SEA-AD neurons (h5ad in data-external/); engine via boundary_detection.R
  external_marker_crosscohort.R       -> independent cross-cohort adjudication of NPTX1/NPTXR/NPTX2
                                         (Dai2026 CSF; Lao2026 4-cohort MTG; Xiao2017 6-region) via P3 Class logic
                                         -> output/tables/external_markers/crosscohort_verdict.csv
  external_marker_panel_audit.R       -> panel adjudication of 14 promising AD markers (GFAP, CHI3L1/YKL-40,
                                         NRGN, SNAP25, VAMP2, TREM2/sTREM2, VGF, SCG2, CNTN2, NPTX1/2/R, PTGDS, HMOX1)
                                         via 4-axis Class logic -> output/tables/external_markers/panel_verdict.csv
  external_marker_audit.R (EXPANDED)  -> SEA-AD per-cell audit of all 14 panel markers, routed by cell type
                                         (Astro: GFAP/CHI3L1/PTGDS/HMOX1; Micro: TREM2; Neuron: NPTX1/2/R,NRGN,
                                         SNAP25,VAMP2,VGF,SCG2,CNTN2) -> sea_ad_panel_verdict.csv
  panel_finalize.R                    -> merge SEA-AD per-cell + cross-cohort literature -> final_panel_verdict.csv
  external_marker_audit_generic.R     -> cohort-GENERIC corrected trajectory audit (donor-level, bin-free
                                         Spearman vs progression, shape, bin-sensitivity). Parameterised for an
                                         INDEPENDENT cohort (e.g., Leng2021 EC GSE147528, Braak axis) -> <COHORT>_panel_verdict.csv
